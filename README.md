Projeto para a disciplina de IA, programa que utiliza prolog para fazer a recomendação de um jogo a partir de métricas impostas pelo usuário utilizando prolog, e fazendo o comparativo como tempo de resposta de uma IA convencional.

Para poder utilizar o recurso de filtrar pela biblioteca steam é necessário uma chave de API de desenvolvedor da steam, que deve ser colocada no arquivo key.json, a biblioteca da conta alvo deve ter visibilidade pública.
